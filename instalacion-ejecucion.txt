1. instalar un entorno virtual en la carpeta de este proyecto Hmi_Web_induma 
2. Ejecutar el entorno virtual dentro de visual studio code
3. ya dentro del entorno virtual ejecutar el siguiente paso
4. pip install -r requirements.txt 
5. en el entorno virtual ejecutar el nombre del archivo que arranca el servidor
6. py main.py
7. ir al navegador web y colocar la ip http://192.168.0.6:5000
8. Debe aparcer hola mundo en el navegador.
